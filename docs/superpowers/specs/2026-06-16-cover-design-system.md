# 视频封面设计系统 — 系列化模板体系

> 适用于：视频号（9:16） + 小红书（3:4）
> 设计基准：小红书 3:4（1242×1656），视频号为扩展适配
> 品牌：港人中山置業通 Jacky

---

## 1. 设计策略

### 系列化模板（方案 B）

**核心思路：** 4 套封面模板对应 4 种内容类型，共享暖金色视觉基因，各自以辨识色区分。观众看到颜色就能识别内容类型。

| 系列 | 内容类型 | 辨识色 | 色值 | 调性 |
|------|---------|--------|------|------|
| 🔥 笋盘速报 | 楼盘讲解 | 赤陶 | #D4875E | 抓眼球、造紧迫感 |
| 📊 数据拆解 | 市场分析 | 海洋蓝 | #6B9EB3 | 专业、可信、数据感 |
| 💡 Jacky观点 | 口播观点 | 鼠尾草绿 | #7B9B6A | 人情味、观点鲜明 |
| ⚠️ 避坑指南 | 警示教育 | 砖红 | #C4554D | 警醒、保护感 |

### 平台适配策略

- **小红书（3:4，1242×1656）**：设计基准。所有封面在此画布上完成核心设计。
- **视频号（9:16，1080×1920）**：以 3:4 核心区域居中，上下扩展暗色渐变区域（#1A1815 → 渐变延续），核心排版不变。

---

## 2. 配色体系

### 基础色板（所有系列共用）

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景 | #1A1815 | 所有封面底色，暖黑 |
| 主强调色 | #C8A052 | 金色分隔线、光效、通用强调 |
| 文字主色 | #F5F0E8 | 标题、数字、正文 |
| 文字次级 | #B5A898 | 副标题、标签文字 |
| 文字三级 | #7A7065 | 品牌条、集数、水印 |
| 底色抬升 | #2C2822 | 卡片、标签背景 |

### 辨识色使用规则

- 辨识色仅用于：系列标签、渐变光晕、金线、核心数字/强调元素
- 辨识色面积 ≤ 整张封面的 15%
- 每张封面最多出现 3 种有彩色（辨识色 + 金色 + 可能的辅助色）
- 背景始终为 #1A1815，不随系列变化

### 渐变规范

- **方向：** 160° 对角（左上到右下偏转）
- **构成：** 辨识色 20%透明度 → 背景 #1A1815
- **过渡比例：** 辨识色占画面 30% 面积后完全过渡到背景色

## 3. 字体层级

| 层级 | 用途 | 字体 | 字号 | 字色 |
|------|------|------|------|------|
| H1 核心数字 | 笋盘价格 / 关键数据 | Georgia | 48–56px | #F5F0E8 |
| H2 主标题 | 封面核心标题 | Georgia | 20–24px | #F5F0E8 |
| H3 副标题 | 补充信息 / 钩子 | Georgia | 14–16px | #B5A898 |
| H4 标签文字 | 系列标签、卖点标签 | PingFang SC | 10–12px | #B5A898 |
| Body 正文 | 较短说明文字 | PingFang SC | 11–13px | #F5F0E8 |
| Caption 品牌 | 品牌条、集数、水印 | PingFang SC | 8–9px | #7A7065 |

**规则：** 一张封面最多同时出现 3 种字号，保持层级清晰。

## 4. 四系列版式结构

### 共享版式基因（所有系列统一）

```
┌──────────────────────────────┐
│ ❶ 系列标签 (8%高)            │  辨识色药丸标签 + 集数
├──────────────────────────────┤
│                              │
│ ❷ 核心内容区 (60–70%高)      │  各系列差异化
│                              │
├──────────────────────────────┤
│        ── 金线分隔 ──         │  2px #C8A052
├──────────────────────────────┤
│ ❸ 辅助信息区 (10–12%高)      │  标签 / 洞察 / 列表
├──────────────────────────────┤
│ ❹ 品牌条 (8%高)              │  港人中山置業通 · Jacky
└──────────────────────────────┘
```

### 🔥 笋盘速报 — "数字炸弹"布局

- **❶ 系列标签**（8%）：左上角赤陶药丸 `🔥 笋盘速报` + 右上角 `EP.XX`
- **❷ 核心数字区**（60%）：超大字号价格（48–56px Georgia）居中，楼盘名（20px Georgia），短卖点描述
- **❸ 标签区**（12%）：3–4 个卖点标签，灰色胶囊，居中排列
- **❹ 品牌条**（8%）：底部居中，细线分隔
- **适用：** 单楼盘深度介绍、价格冲击型内容

### 📊 数据拆解 — "对比卡片"布局

- **❶ 系列标签**（8%）：左上角海洋蓝药丸 `📊 数据拆解` + 右上角 `EP.XX`
- **❷ 钩子标题**（18%）：对比式标题（如"香港 vs 中山买楼成本大对比"），居中
- **❸ 数据对比区**（40%）：左右分屏 VS 结构，灰色卡片容器
- **❹ 核心洞察**（12%）：底部一句话结论，底色抬升卡片
- **❺ 品牌条**（8%）：底部居中
- **适用：** 市场数据分析、港中对比、趋势解读

### 💡 Jacky观点 — "人物+金句"布局

- **❶ 系列标签**（8%）：左上角鼠尾草绿药丸 `💡 Jacky观点` + 右上角 `EP.XX`
- **❷ 人物头像**（20%）：Jacky 照片/插画，圆形裁切，居中，辨识色边框
- **❸ 观点标题区**（50%）：大字观点标题（20–24px Georgia 居中），金线分隔，钩子副标题
- **❹ 品牌条**（8%）：底部居中
- **适用：** 口播观点、个人见解、经验分享

### ⚠️ 避坑指南 — "警告列表"布局

- **❶ 系列标签**（8%）：左上角砖红药丸 `⚠️ 避坑指南` + 右上角 `EP.XX`
- **❷ 警告标题**（22%）：大字标题 + 问题数量（如"买卖合同 3大陷阱"），居中
- **❸ 清单区**（42%）：3–5 条 ✕ 列表，灰色卡片底板，砖红 ✕ 图标
- **❹ 品牌条**（8%）：底部居中
- **适用：** 法律风险、买房陷阱、避坑教育

## 5. 制作流程

### 路径 A：AI 生成 + 后期处理（推荐前期验证用）

**步骤：**
1. 确定内容主题 → 选择对应系列模板
2. 编写 AI 图像生成提示词（含版式约束、配色、文字内容）
3. AI 生成底图 1242×1656
4. 使用 sharp/jimp 脚本后期叠加：
   - 渐变光效增强
   - 金色分隔线（2px #C8A052）
   - 系列标签药丸 + 品牌条（精确像素控制）
   - 文字内容（如需精确控制位置/字号）
5. 视频号版本：上下扩展至 1080×1920

**现有脚本基础：** `image-cards/cover-tool/create-covers.js`

**提示词模板：** 参考 `image-cards/ganghanghui-cover/prompts/` 格式，包含：
- 3:4 竖版画布声明
- 色板约束（辨识色 + 暖黑背景 + 金色强调）
- 构图分区描述
- 文字内容（如 AI 支持文字生成）或标注"无文字区域"

### 路径 B：Remotion Still（推荐量产统一用）

**步骤：**
1. 在 `remotion-realestate/` 中创建 4 个 `<Still>` 组件，每个对应一个系列
2. 组件 Props 接口：
   ```ts
   interface CoverProps {
     series: 'sundip' | 'data' | 'opinion' | 'warning';
     title: string;
     subtitle?: string;
     tags?: string[];
     episodeNumber: number;
     highlightNumber?: string;  // 笋盘系列的价格数字
     comparisonData?: { left: string; right: string };  // 数据系列
     checklist?: string[];  // 避坑系列
   }
   ```
3. 通过 `npx remotion still` 命令行调用渲染：
   ```bash
   npx remotion still SeriesCover \
     --props='{"series":"sundip","title":"港航汇·三房","highlightNumber":"21.8万",...}' \
     output/cover-20260616.png
   ```
4. 可编写批量脚本循环生成多张封面

**优势：** 文字位置/字号/颜色绝对精确，每期完全统一，可批量自动化。

### 推荐节奏

- **前期**（1–2周）：路径 A，快速出图验证视觉方向
- **稳定后**：切换到路径 B，建立 4 个 `<Still>` 组件，批量生产

## 6. 文件规范

### 目录结构

```
image-cards/
├── sundip/              # 笋盘速报系列
│   ├── covers/           # 输出封面 PNG
│   │   └── cover-{楼盘}-{序号}.png
│   └── prompts/          # AI 提示词
│       └── {序号}-cover-{描述}.md
├── data/                # 数据拆解系列
│   ├── covers/
│   └── prompts/
├── opinion/             # Jacky观点系列
│   ├── covers/
│   └── prompts/
├── warning/             # 避坑指南系列
│   ├── covers/
│   └── prompts/
└── cover-tool/          # 后期处理脚本
    └── create-covers.js
```

### 命名规范

- **文件命名：** `cover-{内容关键词}-{序号}.png`
  - 例：`cover-ganghanghui-01.png`、`cover-hk-vs-zs-02.png`
- **提示词命名：** `{序号}-cover-{描述}.md`
  - 例：`01-cover-price-bomb.md`

### 输出规格

| 平台 | 尺寸 | 比例 | 格式 |
|------|------|------|------|
| 小红书 | 1242×1656 | 3:4 | PNG |
| 视频号 | 1080×1920 | 9:16 | PNG |

## 7. 设计检查清单

每张封面生成后检查：

- [ ] 系列标签在左上角，辨识色正确
- [ ] 品牌条在底部，文字 "港人中山置業通 · Jacky"
- [ ] 金线分隔存在（#C8A052, 2px）
- [ ] 辨识色面积 ≤ 15%
- [ ] 字体层级正确（Georgia 标题 / PingFang SC 标签）
- [ ] 最多 3 种字号同时出现
- [ ] 背景为 #1A1815 暖黑
- [ ] 小红书输出 1242×1656，视频号输出 1080×1920
- [ ] 文件名符合命名规范
