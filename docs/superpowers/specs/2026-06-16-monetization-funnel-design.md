# Monetization Funnel Design — 港人中山置业IP变现漏斗

**日期**: 2026-06-16
**状态**: 已批准
**目标**: 内容引流 → PM咨询 → 私域沉淀 → 约看成交（佣金模式）

## 背景

Jacky 运营「港人中山置业通」个人IP，周更3-5条口播视频，覆盖视频号+小红书+YouTube。有流量但全链路转化弱：
- 内容质量参差不齐
- 无标准化PM回复话术
- 无私域跟进体系
- 无数据追踪

## 方案选择

采用 **A+C 混合方案**：内容质量翻身战 + 私域漏斗深耕。

---

## 一、内容金字塔架构

三条平台统一内容体系，每周7条：

| 层级 | 目的 | 时长 | 频率 | 发布节奏 |
|------|------|------|------|----------|
| 🔴 引流轰炸 | 抓眼球、触发恐惧/贪婪 | 15-25秒 | 2条/周 | 周六、日 |
| 🟡 笋盘速报 | 展示货源、建立专业感 | 30-45秒 | 3条/周 | 周一、三、五 |
| 🟢 深度拆解 | 建立信任、筛选高意向 | 60-90秒 | 2条/周 | 周二、四 |

**统一CTA**: 「有兴趣即刻PM我，免费领《2026港人中山买房避坑清单》」
**合规注意**: 视频号、小红书不允许导流微信，CTA只说"PM我"，不出现"微信""加好友"等词。

## 二、脚本结构 & 转化钩子

### 通用三段式骨架

```
[0-3秒]   HOOK — 反常识/痛点/数字冲击
[3-8秒]   问题展开
[8-20秒]  核心信息 — 数据/对比/拆解
[20-25秒] 信任锚点 — "我在中山实地看了XX个盘"
[25-30秒] CTA — "有兴趣即刻PM我"
```

### 三种层级的叠加卡片配置

| | 引流轰炸 (15-25s) | 笋盘速报 (30-45s) | 深度拆解 (60-90s) |
|---|---|---|---|
| 卡片序列 | HookCard → CTA | HookCard → DataPanel → PriceReveal → CTA | HookCard → DataComparison → Checklist → WarningCard → Timeline → CTA |
| Remotion模板 | PipOverlay 轻量模式 | PipOverlay 速报模式 | PipOverlay 深度模式 |

### 5种钩子类型

| 钩子类型 | 文案模板 | 对应组件 |
|----------|----------|----------|
| 恐惧钩子 | "港人在中山买错房的代价，比你想象的大10倍" | WarningCard |
| 数字钩子 | "中山这个盘，港人买了3年，亏了47万" | DataComparisonCard |
| 反常识钩子 | "99%港人不知道，中山最值钱的区不是东区" | RevealMythCard |
| 价格钩子 | "港币100万在中山能买什么？答案颠覆你想像" | PriceRevealCard |
| 限时钩子 | "深中通道通车后，这个盘一个月涨了2000/㎡" | StatCard |

## 四、私域漏斗 & 成交跟进

### 四阶漏斗

```
PM咨询 → 领清单 → 建立信任 → 约看房成交
(第1天)   (第1天)   (第1-7天)   (第2-4周)
```

### 第一阶段：PM即时响应（话术模板）

```
「你好！我係 Jacky，專幫港人喺中山置業。呢份係《2026港人中山買樓避坑清單》，涵蓋：
✅ 港人最常中招嘅5個買樓陷阱
✅ 中山各區真實樓價對比表
✅ 深中通道沿線樓盤升值潛力排名

另外想了解下，你嘅預算同目標區域係？等我可以俾啲針對性建議🙏」
```

### 第二阶段：清单诱饵物料

| 清单内容 | 格式 | 工具 |
|----------|------|------|
| 5大买楼陷阱 | 信息图 | baoyu-infographic |
| 各区真实楼价对比 | 对比表 | baoyu-infographic |
| 深中通道沿线升值排名 | 排名图 | baoyu-infographic |
| 港人买楼流程步骤 | 流程图 | baoyu-infographic |

### 第三阶段：意向分层 & 养熟

| 意向等级 | 判断标准 | 跟进策略 |
|----------|----------|----------|
| 🔥 热 | 问了具体楼盘/价格/流程 | 48小时内私聊推进，约视频带看 |
| 🟡 温 | 领了清单但只说"谢谢" | 3天后发深度拆解视频链接 |
| 🔵 冷 | 领了清单没回复 | 7天后群发笋盘速报激活 |

### 第四阶段：成交推进节奏

```
第1天：PM → 回复+发清单+反问需求
第2天：根据需求推荐1-2个匹配楼盘
第3天：发该楼盘的深度拆解视频
第5天：约视频带看或发实拍素材
第7天：收反馈 → 推进实地看房 → 成交
```

### 每周私域运营节奏

| 日期 | 动作 |
|------|------|
| 周一 | 整理上周新加好友、标注意向等级（微信标签） |
| 周三 | 群发笋盘速报（仅温+冷客户） |
| 周五 | 跟进热客户，推进约看 |
| 周六 | 发朋友圈：成交见证/实地睇楼花絮 |
